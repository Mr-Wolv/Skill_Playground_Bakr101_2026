# TDD (redirect)

This skill is kept as a stable entry point for agents that call `tdd` by name.
Its full content now lives in **[test-driven-development](../test-driven-development/SKILL.md)**.

Use `test-driven-development` when:
- implementing any logic, fixing any bug, or changing any behavior
- proving code works before/after a change
- applying the red-green-refactor loop

Nothing further is maintained here; see the canonical skill for the complete
method, anti-patterns, and loop rules.
