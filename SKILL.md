# Skill Playground

A curated repository of **233 skills** for accelerating AI-assisted software development. These skills span testing patterns, architecture & design principles, DevOps automation, documentation generation, code quality tooling, engineering mindset practices, and much more.

All skills are **bidirectionally synced** between this project's `skills/` directory and your global `~/.agents/skills/` store.

## Quick Start

```bash
# Install all skills from this repo
npx skills add <owner>/skill-playground

# Or install individual skills by name
npx skills add <owner>/skill-playground --skill unit-test-writer
npx skills add <owner>/skill-playground --skill docker-configurator
```
